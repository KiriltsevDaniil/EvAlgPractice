# EvAlgPractice <br />

### Необходимо для запуска: <br />
Предустановить Python 3.9 и PySide2 <br />

### Запуск программы: <br />
Из корневой папки (EvAlgPractice) необходимо запустить файл main.py командой: 

<br /> **python3 main.py** <br />
<br />
